Installation
============

Requirements
------------

- Python == 2.7, 3.4, 3.5, 3.6
- Matplotlib == 2.2, 3.0

Python 3.7 may be available, but it is not checked.


Installation using pip
----------------------
You can install from `PyPI <https://pypi.org/project/mpl-axes-aligner/>`_::

  pip install mpl-axes-aligner
